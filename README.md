# Othello (Reversi) — WPF (C# / .NET)

A WPF implementation of the board game **Othello (Reversi)** built in **C#/.NET**.  
This project was created as a course assignment focusing on **GUI development**, **OOP**, and a clean separation of responsibilities (MVVM-style structure).

## Features
- 8×8 Othello board with valid move handling and disk flipping
- Turn-based gameplay with game-over detection
- Supports different player types (e.g. Human / Computer) *(depending on setup)*
- Clear project structure with separate folders for Models, Views, ViewModels, and Controllers

## Tech
- C# / .NET
- WPF (XAML)
- MVVM-style structure (Views + ViewModels + Models)

## How to run
### Visual Studio
1. Open `Othello.sln` in Visual Studio
2. Press **Start** (or `F5`) to run
